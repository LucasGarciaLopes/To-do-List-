# To-Do List com JavaScript

Aplicação de lista de tarefas desenvolvida com JavaScript puro, manipulando DOM e utilizando LocalStorage para salvar os dados.

## 🔧 Tecnologias
- HTML5
- CSS3
- JavaScript
- DOM
- LocalStorage

## 💻 Funcionalidades
- Adicionar tarefas
- Remover tarefas
- Marcar como concluída
- Persistência de dados

## 🚀 Como visualizar
Abra o arquivo `index.html` no navegador.
